The Data

Our dataset in this lab concerns baby names and their popularity over time. The dataset babies_ca has information about baby names in California from 1940-2016. For each year, and for each name with at least 50 recorded babies born, we are given the counts of how many babies had that name.

babies_ca <- read.csv(here::here("8_Regression_Testing", "Lab-regression-babynames", "babies_ca.csv"))

At this link, you can find the names for ALL 50 states, in separate datasets organized by first letter.

Is my name not cool any more?

Lets take a look at how the name Kelly has changed over time.

  1. Make a plot showing the number of babies named Kelly born each year in CA.
    Color the plot to show the gender assigned at birth to these babies.
babies_ca %>%
  filter(Name == "Kelly") %>%
  ggplot(aes(x = Year, y = Count, color = Gender)) +
  geom_line()

  1. I was born in 1989. Lets look at only the time frame since I was named. Narrow down the dataset to only 1989 onward.
kellys_89 <- babies_ca %>%
  filter(Name == "Kelly") %>%
  filter(Year >= 1989)
  1. Create a linear model with the year as the explanatory variable, and the number of Kellys as the response. Summarize the results, and plot the model.
my_model <- lm(Count ~ Year, data = kellys_89)

  1. Plot the residuals: that is, the actual values minus the predicted values.
    (The function add_predictions in the modelr package may be useful to you.)

Comment on the residuals - do you see any patterns?

kellys_89 <- kellys_89 %>%
  add_predictions(my_model) %>%
  mutate(
    residual = Count - pred
  )

The residuals follow a clear pattern, with half the observations overestimating
and half underestimating.  This is because we did not account for the gender
of the babies.

Spelling by state

I used to hate it when people would spell my name as Kelli or Kelley. But I dont have it as bad as my good friend Allan.

  1. Narrow the California dataset down to only male-assigned babies named Allan, Alan, or Allen. Make a plot comparing the popularity of these names over time.
babies_ca %>%
  filter(Gender == "M", Name %in% c("Allan", "Alan", "Allen")) %>%
  ggplot(aes(x = Year, y = Count, color = Name)) +
  geom_line()

  1. In California, Allans spelling of his name is the least common of the three - but perhaps its not such an unusual name for his home state of Pennsylvania. Compute the total number of babies born with each spelling of Allan in 2000, in Pennsylvania and in California.
a_names <- read.csv(here::here("8_Regression_Testing", "Lab-regression-babynames", "StateNames_A.csv"))

allans <- a_names %>%
  filter(Name %in% c("Allan", "Alan", "Allen"),
         State %in% c("CA", "PA"),
         Year == 2000) 

allan_counts <- allans %>%
  group_by(Name, State) %>%
  summarize(Count = sum(Count))

  1. Perform a Chi-Square test on this data, to determine if CA and PA have different distributions of spellings of Allan.
allan_counts %>%
  ungroup() %>%
  pivot_wider(names_from = State, values_from = Count) %>%
  select(-Name) %>%
  chisq.test()
